### How to Use This Template
1.  **Replace Bracketed Text:** Fill in everything inside `[ ]` with your specific details.
2.  **Tables:** Ghostwriter renders the markdown tables automatically in the preview pane. You can edit the text inside the table cells directly.
3.  **Code Blocks:** Any text inside triple backticks \`\`\` will be highlighted as code in the preview.
4.  **Export:** When finished, go to **File → Export → PDF** (or HTML) to generate a clean, readable report for sharing.
